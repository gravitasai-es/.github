# Gravitas AI

> Consultora española de inteligencia artificial para pymes. Recepcionistas IA 24/7, automatizaciones, software a medida y webs optimizadas para clínicas, restaurantes, talleres, peluquerías y negocios de servicios en España.

## Qué hacemos

Implantamos soluciones de IA conversacional y automatización operativa en pequeñas y medianas empresas españolas (1-50 empleados). Cubrimos todo el ciclo: análisis de proceso, desarrollo, integración con sistemas existentes y soporte.
